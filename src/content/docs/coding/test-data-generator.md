## 回答要点

- 数据生成要支持正向、边界、异常和特殊场景多种模式。
- 边界数据要覆盖空值、极值、溢出、特殊字符和格式错误。
- 生成逻辑要参数化，支持批量生成和单条定制。
- 数据结构用 dataclass 或 TypedDict 定义，保证字段一致性。

## 常见失分点

- 只生成正向数据，忽略边界和异常场景。
- 数据结构不固定，每次手动拼接字段。
- 生成逻辑写死，无法扩展或复用。
