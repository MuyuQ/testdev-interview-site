## 定义

API 断言是在接口测试中验证响应数据是否符合预期的语句。

## 为什么重要

断言是自动化测试的核心，没有断言的测试无法自动发现问题。

## 常见错误

- 只验证状态码，不验证响应体
- 断言过于具体，导致测试脆弱
- 缺少对边界条件的断言

## 高频问题

### 接口测试中应该断言什么？

1. 状态码
2. 响应时间
3. 关键字段值
4. 数据结构
5. 业务逻辑
